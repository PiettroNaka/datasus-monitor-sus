O Portal DATASUS TabNet

O TabNet é uma ferramenta genérica de tabulação de dados que permite gerar tabelas a partir de arquivos DBF, amplamente utilizada pelo Ministério da Saúde para disseminar informações do SUS.

É a principal interface para consulta de dados de produção hospitalar (SIH) e ambulatorial (SIA), essencial para a gestão pública de saúde no Brasil.

Desafios Técnicos

Navegação complexa baseada em frames e scripts legados.
Instabilidade em consultas de grande volume de dados.
Saída de dados com metadados que exigem limpeza rigorosa.
